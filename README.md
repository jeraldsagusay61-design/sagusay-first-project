# Jerald Sagusay - Personal Portfolio

A modern, responsive, and aesthetic personal portfolio website showcasing skills, projects, and contact information.

## 🌟 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Aesthetics**: Dark theme with gradient backgrounds and smooth animations
- **Multiple Pages**: Home, About, Projects, and Contact pages
- **Professional Styling**: Clean CSS with hover effects and transitions
- **Profile Integration**: Integrated profile photo and personal information
- **Contact Information**: Easy ways to get in touch via phone and email

## 📄 Pages

1. **Home** - Welcome section with quick overview and skills showcase
2. **About** - Detailed information about me, technical skills, and journey
3. **Projects** - Showcase of projects and project categories
4. **Contact** - Contact information and ways to connect

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with gradients and animations
- **JavaScript** - Interactive elements
- **Python Flask** - Backend framework (optional)
- **GitHub Pages** - Hosting
